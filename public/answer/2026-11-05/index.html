📖 Meaning medium

琳琅满目 (lín láng mǎn mù)

A vast array of beautiful and exquisite things; a feast for the eyes.

Literal: Beautiful jade fills the eyes

💡 Example

The museum's collection of ancient jade artifacts was so vast that it was truly a case of 'linlang manmu'.

🎯 Hints (for next time)

Hint 1 — Subtle

This idiom describes an overwhelming visual abundance of high-quality items.

Hint 2 — More Obvious

The second character refers to a type of precious gemstone.
